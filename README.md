# 👥 客户备注本

一个简单易用的客户备注管理工具，支持多类型记录、导入导出、模糊搜索等功能。

## ✨ 功能特点

- 🔐 **登录保护** - 需要账号密码才能访问
- 👥 **客户管理** - 用编号管理客户信息
- 📝 **多类型记录** - 支持订单、备注、内容、跟进、其他
- 🔍 **模糊搜索** - 搜索编号、名称、内容
- 📋 **一键复制** - 快速复制记录内容
- 📥📤 **导入导出** - 支持 Excel 和 TXT 格式
- 💾 **本地存储** - 数据保存在浏览器中

## 🛠️ 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
